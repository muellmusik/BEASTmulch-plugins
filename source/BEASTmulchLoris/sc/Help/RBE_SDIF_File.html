RBE_SDIF_File represents a SDIF file with RBEP Frames


Inherits from: Object : AbstractFunction : Stream : IOStream : UnixFILE : File


RBE_SDIF_File represents a SDIF file with RBEP Frames, which contain partial based on the Loris Reassigned Bandwidth-Enhanced Additive Sound Model.


RBEL frames (and any other types) are currently ignored.


See also: Loris, BEPartials, BEPartial, BEOsc


Creation / Class Methods


*new (path)

Open a SDIF File and return a new RBE_SDIF_File. As with its superclass, one can use the method isOpen to see if this was successful.

path - A String containing the path of the SDIF file to open.

f = RBE_SDIF_File("A.sdif".resolveRelative); // open a SDIF file with RBEP frames

f.isOpen;

f.close;

f = RBE_SDIF_File("foo"); // try to open a non-existant file

f.isOpen;

f.close;



Instance Methods


readFramesToPartials

Returns an Array of Arrays of partial breakpoint data in the form [[startTime, times, freqs, phases, bandwidths, amps] * number of partials].

f = RBE_SDIF_File("A.sdif".resolveRelative); // open a SDIF file with RBEP frames

f.readFramesToPartials;

f.close;
