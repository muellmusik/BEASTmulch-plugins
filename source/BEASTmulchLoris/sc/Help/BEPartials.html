BEPartials a list of Loris Bandwidth Enhanced Partials


Inherits from: Object


BEPartials represents a list of partials in the Loris Reassigned Bandwidth-Enhanced Additive Sound Model. Its individual partials are instances of BEPartial.


See also: Loris, BEOsc, BEPartial, RBE_SDIF_File


Creation / Class Methods


*new (sdif)

Create a new BEPartials.

sdif - An instance of RBE_SDIF_File pointing to a SDIF file with RBEP (Loris) partial frames. 

f = RBE_SDIF_File("A.sdif".resolveRelative); // open a SDIF file with RBEP frames

p = BEPartials(f);

f.close;



Accessing Instance and Class Variables

partialList_

partialList

Set or get this object's partial list. This is an Array containing instances of BEPartial.

Instance Methods


at (index)

Returns the BEPartial at index.



size

Returns the number of partials in this list.



dur

Returns the duration of this list in seconds.



fadeInOut

Checks partials for non-zero amplitudes at start and end breakpoints and adds additional fadein and out breakpoints if needed.



ar (stretch, pitch, bw)

Returns an Array of Arrays of three EnvGens and a LorisPhaseGen (one Array for each partial) suitable for use as the inputs to a BEOsc. The order for each partial is [freq, phase, bw, amp]. The arguments below may be constants, control rate UGens, or Functions. If the latter, the Function will be evaluated separately for each partial. (See examples below.)

stretch - A time stretch factor. Higher is longer. Default value is 1.

pitch - A pitch scale factor. Default value is 1.

bw - A bandwidth scale factor. Default value is 1.



Examples



s.boot;

f = RBE_SDIF_File("A.sdif".resolveRelative); // open a SDIF file with RBEP frames

p = BEPartials(f);

f.close;



(

SynthDef(\A, { arg stretch = 1, pitch = 1, bw = 1, mul = 1, gate = 1, attack=0, release=0.1, out=0;

var outmix, envs;


// returns [[freqEnvGen, a LorisPhaseGen, bandwidthEnvGen, ampEnvGen] * numPartials]

envs = p.ar(stretch, pitch, bw); 


// pan each partial and mix

outmix = Mix(PanAz.ar(2, BEOsc.ar(*envs), {Rand(-1, 1)} ! p.size)) 

* EnvGen.ar(Env.linen(attack, p.dur * stretch, release), gate, mul, doneAction: 2);

Out.ar(out, outmix); 

}).load(s, dir: "".resolveRelative);

)
