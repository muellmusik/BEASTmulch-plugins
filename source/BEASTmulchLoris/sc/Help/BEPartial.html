BEPartial a Loris Bandwidth Enhanced Partial


Inherits from: Object : Collection : SequenceableCollection : ArrayedCollection : Array


BEPartial represents a single partial in the Loris Reassigned Bandwidth-Enhanced Additive Sound Model. Normally you would not create these directly, rather an instance of BEPartials would create them for you.


This is essentially a subclass of Array with a few convenience methods added. Note that when modifying any of the partials elements, it is best practice to reassign back to the partial in case a new Array is returned, for example:


myPartial.times = myPartial.times.squared;


See also: Loris, BEPartials, BEOsc


Creation / Class Methods


*new

Create a new BEPartial.



*newFrom (array)

Create a new BEPartial from an Array of breakpoint data.

array - The breakpoint data, in the form [startTime, times, freqs, phases, bandwidths, amps]. All but startTime are Arrays, and they should be of the same size, except for times, which should be one smaller as its values indicate the duration of segments between partial breakpoints.
